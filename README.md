"Молодая гвардия — время выбора"
**Жанр:** Нарративная историческая игра (Visual Novel / Text Quest)
**Движок:** Ink
## Описание
Игра посвящена подвигу подпольной антифашистской организации «Молодая гвардия», действовавшей в оккупированном Краснодоне во время Великой Отечественной войны.
Вам предстоит принимать сложные решения в условиях оккупации, когда цена каждого шага — жизнь.
## Структура проекта
Сценарий разделен на главы (уровни), которые связаны в главном файле «Молодая гвардия»:
* **Главы 01-08:** Основные события от тишины оккупированного города до арестов и бессмертия подвига.
## Как запустить
Для запуска игры загрузите файлы и откройте главный файл через редактор [Inky](https://www.inklestudios.com/ink/).
